# Pre-V2 archive

The files below record earlier proposals, chat-derived reviews, and the
experimental database design. They are intentionally preserved as history.

They are not authoritative. In particular, they contain obsolete assumptions
about floating-point quantities, duplicated commercial and inventory lines,
trigger-driven posting, mutable recipes, persisted drafts, item-to-item
conversions, and service compatibility adapters.

Use [`../../README.md`](../../README.md) to find the current product and domain
contract.
